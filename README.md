# Mini-Wiki

A lightweight educational website presenting clear and concise explanations of selected science-fiction concepts.  
This project was created as part of an academic web design assignment to demonstrate structured HTML, clean layout, and simple navigation.

---

## Overview

Mini-Wiki is a small static website that simplifies complex sci-fi topics into short, readable articles.  
Each page follows a consistent layout, uses dropdown-style sections, and maintains a clean dark theme for readability.

---

## Topics Included

- **Alien Life** — Scientific search for extraterrestrial organisms  
- **Time Travel** — Relativity, paradoxes, and theoretical ideas  
- **Simulation Theory** — The possibility that reality may be simulated  
- **Parallel Universe** — Multiverse concepts and alternate realities  
- **Technological Singularity** — Artificial intelligence and future human–technology merging

- Consistent dark theme  
- Expandable dropdown sections  
- Easy navigation across pages

## Usage

1. Download or clone the repository  
2. Open `index.html` in a browser  
3. Navigate between topic pages using the menu  
4. Submit feedback through the feedback page
